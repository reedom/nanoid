```
NAME:
   nanoid - Generate a new random ID(by nanoid library)

USAGE:
   nanoid [global options]

EXAMPLE:
   nanoid
     -> vtOidCnXVkbZNL2POqgtq
   nanoid --prefix image/ --suffix .png
     -> image/vtOidCnXVkbZNL2POqgtq.png

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --prefix value, -p value  Add `value` before the ID
   --suffix value, -s value  Add `value` after the ID
   --help, -h                show help
```
